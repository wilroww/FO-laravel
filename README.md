# FO-laravel
Set up:
Make sure php,laravel,composer,vscode is already in system path.
1. Download Zip file (FO-laravel)
2. Extract Zip file
3. Open Vscode
4. Click file, then open folder, then navigate where FO-laravel file is, then select folder.
5. Click terminal, click new terminal.
6. On terminal run: composer install
7. Then run: php artisan migrate
8. Then run: php artisan db:seed
9. Then run: php artisan serve
10. Click its http, then its done.
